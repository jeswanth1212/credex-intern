## Features Implemented

- **Hero Section**: Headline, subheading, and two CTA buttons ("Sell My Licenses", "Get a Quote")
- **How It Works**: Three-step process (Upload License → Get Valuation → Get Paid) with icons
- **Why Choose Us**: Four bullet points/tiles with icons and descriptions
- **Customer Testimonials**: Two dummy reviews (name, role, company)
- **Contact/Lead Form**: Name, Email, Company, License Type (dropdown), Message, with frontend validation only
- **Modern UI/UX**: Clean, professional, visually appealing, and mobile responsive
- **Tailwind CSS**: Used for all styling
- **SEO Meta Tags**: Title, description, keywords, favicon
- **Light/Dark Mode Toggle**: Theme switcher with system preference detection
- **Animations**: Framer Motion for smooth transitions and section reveals
- **Logo Placeholder & Favicon**: Custom SS logo in navbar and favicon
- **AI Chat Widget**: Mock LLM-powered chat widget with example questions and fallback responses

## Tech Stack

- **Framework**: React.js + Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide, SVG
- **Deployment Ready**: Vercel
- **No backend required**: All data is static/dummy

---

## Design Choices

- **Color Palette**: Teal/emerald gradient for a fresh, trustworthy feel
- **Typography**: Inter font, clear hierarchy
- **Iconography**: Consistent, modern icons
- **Accessibility**: Semantic HTML, proper contrast
- **Responsiveness**: Mobile-first, grid/flex layouts

## Setup Instructions

1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```
3. Run the development server:
   ```
   npm run dev
   ```
